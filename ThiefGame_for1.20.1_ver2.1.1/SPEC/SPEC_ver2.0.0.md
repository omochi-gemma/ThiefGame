# 🎭 職業追加・調整

## 🟥 新職業「復讐者 (Avenger)」
ハイリスク・ハイリターン。扱いが難しいが火力は随一。

### パッシブ（実装済み） ✅
被弾するたびに「復讐心」が蓄積。死亡するとリセット。  
- 復讐心 5: 跳躍力上昇 II  
- 復讐心 10: 上記 + 移動速度上昇 I  
- 復讐心 15: 上記 + 耐性 I + 再生 I  

### スキル
- **アドレナリン（adrenaline）** ✅  
  - 一瞬無敵  
  - 無敵中に攻撃されなかった場合は自分がダウン  
- **憤怒の心（wrathful_heart）** ✅  
  - 周囲の館の主をスタン  
  - 復讐心に応じて範囲・時間上昇  
  - 0: 範囲1 / 1秒  
  - 5: 範囲3 / 3秒  
  - 10: 範囲5 / 5秒  
  - 15: 範囲10 / 10秒  

---

## 🔵 新職業「時間魔術師 (Chronomancer)」
ローリスク・ローリターン。操作が簡単で汎用性高。

### パッシブ
- 時間停止中、移動速度上昇・透明化

### スキル
- **月の砂時計（moon_hourglass）**  
  - 時間停止  
  - 館の主をスタン＆スキル封印  
  - 制限時間も停止  
  - エリア進行（鍵開け等）は不可  
  - 使用するたび持続時間が短縮（最短 1 秒）

- **太陽の懐中時計（sun_pocket_watch）**  
  - 所持中、泥棒・館の主ともに  
    - 移動速度上昇  
    - スキル CT 短縮  
  - 館の主がデバフ状態のときに有効

### 共通パッシブ

### 能力一覧

#### **聡明**
- 血痕右クリックで泥棒全員に発光、自身に透明化  
- 相性：〇 忍者 / ー 技術者・看護師・復讐者・時間魔術師 / ✖ 超能力者

- スニークでジャンプすると短時間超移動速度  
- 相性：〇 超能力者・復讐者 / ー 看護師・時間魔術師 / ✖ 忍者・技術者

#### **逆境**
- 序盤は弱いがエリア進行で強くなる  
- 相性：〇 看護師・時間魔術師 / ー 忍者・技術者・超能力者 / ✖ 復讐者

#### **博徒**
- 一定時間ごとに攻撃力が変動  
- パーティクルで双方に可視化  
- 相性：〇 復讐者 / ー 忍者・技術者・超能力者・看護師 / ✖ 時間魔術師
