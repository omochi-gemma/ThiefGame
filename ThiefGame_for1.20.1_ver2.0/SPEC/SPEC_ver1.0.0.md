『謎解き』
謎1 単純な探索。鍵の持ちてと先端を見つけて、作業台の上に投げると宝物庫への鍵を作成できる。宝物庫には脱出に必要なレッドストーンブロックとがある。
屋上にある2階のカギを入手さえすれば先へは進める。
1階と3階と屋上のみ行き来できるため、簡単な謎にする。

謎2 ディスコードで「なれっと パスワード何？」と入力すると3桁の数字を教えてくれる。その数字を入力すると魔法のガラス(別館への渡り廊下を修復する)が手に入る。
348

謎3 6色のシュルカーボックスがある。6桁の番号を並べることで謎が解ける。シュルカーボックスを開けると左上に文字が書いてあり、赤→青→黄→緑→紫→橙の順に入力する。青と黄と紫は逆さまに設置されており、6とEが書かれているが、逆さまなため、正しくは9と3である。
謎を解くと白色ガラスを破壊できるツールを入手できる。
G86U3P

謎4 6色のコンクリートパウダーを、同じ色が隣り合わせにならず、対角線にもないように設置する。コンクリートは謎3で見つけたシュルカーボックスに入っている。

謎5 謎1で手に入れたコアを最奥部に設置し、レバーを倒すと装置が起動して上へ浮遊することができる。浮遊で上に行くことで脱出成功となり、泥棒側が全員脱出か死亡するとゲーム終了となる。

『ゲーム性』
館の主と泥棒チームに別れて、館の主は泥棒の全滅、泥棒チームは脱出を目指す。
泥棒チームは体力が40あり、体力が20以下になるとダウン状態になる。
ダウン状態になると、体力増強エフェクトが切れ、強制的に這って移動することになり、はしごや階段、段差を登れない。
ダウン状態では5秒毎に2ダメージ受け、死亡すると残機が減り、残機が0になると観戦になる。残機はデフォルトで2であり、変更可能。
館のチェストには、パン、クールタイム減少玉、治癒のスプラッシュポーションがランダムに入っている。



「館の主」
館の主はげんこつ(左クリック)で7ダメージ与えることができる。
シフト中は空腹度を回復できるが、それ以外のときは空腹が付与されてしまう。
常に移動速度低下が付与される。
ダウン中の泥棒を攻撃することはできない。

体力が半分以下(20以下)になるとダウン状態に移行する。ダウン状態になると体力増強が消え、一旦体力が20になる。
ダウン状態では頭上に透明なシュルカーが設置され、常にクロール状態になる。(シュルカーの顔はリソースパックで魂のような見た目にする。)
ダウン状態の味方へは、泥棒チームにのみ見えるパーティクルがビームのように表示される。
魂をダウン状態でない泥棒チームが右クリックすると蘇生が進み、5秒間右クリックすると蘇生が完了する。この際蘇生されたプレイヤーは体力が最低21の状態で復活する。
ダウン状態のプレイヤーは5秒ごとにダメージを受け、体力が無くなると完全に死亡し、残機を消費して復活する。残機が無くなれば二度と復活できなくなる。※ダウン中でもアイテムは使えるため、延命が可能。
